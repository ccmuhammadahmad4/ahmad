# Monitoring & Alerting Stack (Prometheus + Grafana)

## Project Overview
This project sets up a monitoring and alerting stack using Prometheus and Grafana to observe application metrics and system performance.

## Tools Used
- Prometheus
- Grafana
- Docker
- Kubernetes

## Setup Instructions
1. Install Docker or Kubernetes cluster
2. Deploy Prometheus and Grafana using Docker Compose or Helm
3. Configure Prometheus scrape targets

## Implementation Steps
- Monitor sample app metrics (CPU, memory, HTTP requests)
- Create Grafana dashboards for visualization
- Set up alerting rules in Prometheus
- Configure Alertmanager for notifications

## Bonus Tasks
- Add Loki for log aggregation
- Integrate Tempo for distributed tracing
- Create custom exporters for business metrics
