1. Grammar: Question Formation
- Learn how to form questions using auxiliary verbs (do/does/did/is/are/will/can).
- Examples:
  - Do we need to restart the server?
  - Is the deployment complete?
  - Can you check the logs?

Practice:
Write 3 questions you might ask during a deployment or team meeting.

2. Vocabulary: DevOps and Workplace Terms

| Word         | Meaning                          | Example Sentence                         |
|--------------|----------------------------------|------------------------------------------|
| **Trigger**  | To start a process               | We trigger the pipeline after each commit. |
| **Alert**    | Notification of an issue         | The system sent an alert about high CPU usage. |
| **Downtime** | Period when service is unavailable | We had 2 hours of downtime last night. |
| **Patch**    | Software update to fix issues    | The patch resolved the DNS error. |
| **Latency**  | Delay in processing               | The latency increased after the last release. |

Practice:
Use each word in a sentence related to your work.
